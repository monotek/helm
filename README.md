# Zammad Helm chart

A [Helm](https://helm.sh) chart to install [Zammad](https://zammad.org) on [Kubernetes](https://kubernetes.io)

## Add the Helm repo via

```console
helm repo add zammad https://zammad.github.io/zammad-helm/
```
